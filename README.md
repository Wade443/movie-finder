## MOVIE FINDER

A simple movie search web app built using HTML, CSS, and JavaScript.
It uses the OMDb API to fetch and display movie data dynamically.

## FEATURES

-Search for movies by title
-Display movie results with posters and basic info
-Click on a movie to see more details (plot, genre, rating)
-Featured movie section that updates automatically
-Responsive design for mobile and desktop

## TECHNOLOGIES USED

HTML
CSS
JavaScript
Fetch API (async/await)
OMDb API
